# Refactory Academy Website Clone

A clone of the Refactory Academy landing page built with HTML and CSS.

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox

## Features

- Responsive design (mobile, tablet, desktop)
- Modern layout using CSS Grid and Flexbox
- Hover effects on buttons and links
- Clean and organized code with comments

## Sections Included

1. Navigation Bar
2. Hero Section
3. About Refactory Section
4. Featured Courses Section
5. Footer Section

## Layout Techniques

- **CSS Grid**: About section (2 columns), Courses (3 columns), Footer (4 columns)
- **Flexbox**: Navbar, Hero buttons, Social icons
